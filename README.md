# Difference Generator

Difference Generator allows you to compare 2 files and output the difference in one of the formats (string, yaml, plain).

## Requirements

MacOS, Linux, Windows

## Installation

```bash
make install
```

## Usage
Generating difference of 2 files, format not specified.
